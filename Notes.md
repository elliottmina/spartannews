# Spartan news

## Collector features
* Allow sources to define filters
* Analyzes for keywords. 

### serving
* Fetch time slice
* Fetch all
* fetcher headers
* fetch sections
* fetch sources
* fetch keywords

### source type
* index (new site, HN)

### filters
* Blacklist authors
* Blacklist websites
* Blacklist keywords

## client features
* Display content
* Track last requested date so we can get what is fresh
* Record what items have been read
  * purge from server
  * add and configure sources
    * preview configuration effects
